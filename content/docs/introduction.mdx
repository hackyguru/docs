# Introduction

Welcome to our documentation framework! This modern documentation site builder is built with Next.js, MDX, Tailwind CSS, and other powerful tools to help you create beautiful, searchable, and customizable documentation.

## Example Components

### Alerts

<Alert>
  This is a default alert
</Alert>

<Alert variant="destructive">
  This is a destructive alert
</Alert>

### Code Blocks

```js filename="example.js"
function greet(name) {
  return `Hello, ${name}!`
}

console.log(greet('World'))
```

### Tabs

<Tabs defaultValue="npm">
  <TabsList>
    <TabsTrigger value="npm">npm</TabsTrigger>
    <TabsTrigger value="yarn">yarn</TabsTrigger>
    <TabsTrigger value="pnpm">pnpm</TabsTrigger>
  </TabsList>
  <TabsContent value="npm">
    ```bash
    npm install
    ```
  </TabsContent>
  <TabsContent value="yarn">
    ```bash
    yarn install
    ```
  </TabsContent>
  <TabsContent value="pnpm">
    ```bash
    pnpm install
    ```
  </TabsContent>
</Tabs>

### Cards

  <CardHeader>

  </CardFooter>
