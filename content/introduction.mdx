# Introduction

Welcome to our modern documentation framework built with Next.js and MDX. This framework provides a beautiful, responsive, and feature-rich environment for creating and maintaining documentation.

## Key Features

- **MDX Support**: Write documentation using Markdown with JSX components
- **Beautiful UI**: Clean and modern interface with light/dark mode support
- **Full-text Search**: Built-in search functionality for your documentation
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Code Highlighting**: Syntax highlighting for code blocks with copy functionality
- **Navigation**: Customizable sidebar, breadcrumbs, and "On This Page" navigation
- **Media Support**: Embed images and YouTube videos in your documentation
- **Customizable**: Fully configurable through a central configuration file

## Key Concepts

### Configuration

The framework is highly configurable through the `docsConfig.js` file. You can customize:

- Site metadata and branding
- Navigation structure
- Theme settings
- Footer content
- And more

### Content Organization

Documentation is written in MDX and stored in the `content/docs` directory:

```txt filename="Directory Structure"
content/
  docs/
    introduction.mdx
    installation.mdx
    features/
      mdx.mdx
      components.mdx
    guides/
      adding-pages.mdx
```
