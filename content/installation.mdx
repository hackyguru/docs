# Installation

This guide will walk you through setting up your documentation site from scratch.

## Prerequisites

Before you begin, make sure you have:
- Node.js 18.0 or later
- npm, yarn, or pnpm package manager
- Basic knowledge of React and Next.js

## Quick Start

### 1. Create a New Project

Start by creating a new Next.js project:

```bash
npx create-next-app@latest my-docs --typescript --tailwind --app
cd my-docs
```

### 2. Install Dependencies

Install the required dependencies:

```bash
npm install @radix-ui/react-icons lucide-react next-themes @radix-ui/react-scroll-area @radix-ui/react-separator @radix-ui/react-slot clsx tailwind-merge @radix-ui/react-dialog framer-motion next-mdx-remote gray-matter
```

### 3. Configure Next.js

Update your `next.config.js`:

```js
/** @type {import('next').NextConfig} */
const nextConfig = {
  pageExtensions: ['js', 'jsx', 'ts', 'tsx', 'md', 'mdx'],
  images: {
    remotePatterns: [
      {
        protocol: 'https',
        hostname: '**',
      },
    ],
  },
}

module.exports = nextConfig
```

### 4. Configure Tailwind CSS

Update your `tailwind.config.js`:

```js
/** @type {import('tailwindcss').Config} */
module.exports = {
  darkMode: ["class"],
  content: [
    './pages/**/*.{ts,tsx,js,jsx,mdx}',
    './components/**/*.{ts,tsx,js,jsx}',
    './app/**/*.{ts,tsx,js,jsx}',
    './content/**/*.{md,mdx}',
  ],
  theme: {
    extend: {
      colors: {
        border: "hsl(var(--border))",
        background: "hsl(var(--background))",
        foreground: "hsl(var(--foreground))",
        primary: {
          DEFAULT: "hsl(var(--primary))",
          foreground: "hsl(var(--primary-foreground))",
        },
        secondary: {
          DEFAULT: "hsl(var(--secondary))",
          foreground: "hsl(var(--secondary-foreground))",
        },
        muted: {
          DEFAULT: "hsl(var(--muted))",
          foreground: "hsl(var(--muted-foreground))",
        },
        accent: {
          DEFAULT: "hsl(var(--accent))",
          foreground: "hsl(var(--accent-foreground))",
        },
      },
      borderRadius: {
        lg: "var(--radius)",
        md: "calc(var(--radius) - 2px)",
        sm: "calc(var(--radius) - 4px)",
      },
    },
  },
  plugins: [require("@tailwindcss/typography")],
}
```

### 5. Add Base Styles

Create or update your `globals.css`:

```css
@tailwind base;
@tailwind components;
@tailwind utilities;
 
@layer base {
  :root {
    --background: 0 0% 100%;
    --foreground: 240 10% 3.9%;
    --primary: 240 5.9% 10%;
    --primary-foreground: 0 0% 98%;
    --secondary: 240 4.8% 95.9%;
    --secondary-foreground: 240 5.9% 10%;
    --muted: 240 4.8% 95.9%;
    --muted-foreground: 240 3.8% 46.1%;
    --accent: 240 4.8% 95.9%;
    --accent-foreground: 240 5.9% 10%;
    --border: 240 5.9% 90%;
    --radius: 0.5rem;
  }
 
  .dark {
    --background: 240 10% 3.9%;
    --foreground: 0 0% 98%;
    --primary: 0 0% 98%;
    --primary-foreground: 240 5.9% 10%;
    --secondary: 240 3.7% 15.9%;
    --secondary-foreground: 0 0% 98%;
    --muted: 240 3.7% 15.9%;
    --muted-foreground: 240 5% 64.9%;
    --accent: 240 3.7% 15.9%;
    --accent-foreground: 0 0% 98%;
    --border: 240 3.7% 15.9%;
  }
}

@layer base {
  * {
    @apply border-border;
  }
  body {
    @apply bg-background text-foreground;
  }
}
```

### 6. Create Configuration

Create `config/docs.js` in your project:

```js
export const docsConfig = {
  site: {
    title: 'Your Documentation',
    description: 'Documentation site built with Next.js',
  },
  theme: {
    accentColor: '#0070f3',
    font: {
      sans: 'Inter',
      mono: 'JetBrains Mono',
    },
  },
  sidebar: {
    groups: [
      {
        title: 'Getting Started',
        items: [
          {
            title: 'Introduction',
            href: '/introduction',
          },
          {
            title: 'Installation',
            href: '/installation',
          },
        ],
      },
    ],
  },
}
```

### 7. Start Development Server

Run the development server:

```bash
npm run dev
```

Visit `http://localhost:3000` to see your site.

## Project Structure

After installation, your project structure should look like this:

```txt
my-docs/
├── app/               # Next.js app directory
├── components/        # React components
├── content/          # Documentation content (MDX)
├── config/           # Configuration files
├── lib/              # Utility functions
├── public/           # Static assets
├── styles/           # CSS styles
├── next.config.js    # Next.js configuration
└── tailwind.config.js # Tailwind configuration
```

## Next Steps

Now that you have your documentation site set up:

2. Add MDX content in the `content` directory
3. Customize the theme and components
4. Deploy your site
